# Telegram-бот «Какое у вас тотемное животное?»

Проект на Python и aiogram 3. Бот проводит пользователя по викторине, считает баллы за ответы и показывает животное Московского зоопарка как результат. Сделано по заказу Московского зоопарка

## Что умеет бот

- запускается командой `/start`;
- показывает приветствие и кнопку начала викторины;
- задаёт вопросы с несколькими вариантами ответа;
- считает баллы по животным;
- показывает итоговое животное, описание, факт и блок про программу опеки;
- отправляет изображение животного из папки `bot/animals/`;
- даёт текст для соцсетей в копируемом блоке;
- даёт ссылку на контактное лицо `.env` и предлагает готовое приветственное сообщение;
- сохраняет отзывы пользователей в `reviews.txt`.

## Установка

Создайте виртуальное окружение:

```bash
python -m venv venv
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Создайте в корне проекта файл `.env` и заполните переменные по примеру:

```env
BOT_TOKEN=1234567890:replace_with_your_telegram_bot_token
CONTACT_PROFILE_URL=https://t.me/username_сотрудника
BOT_LINK=https://t.me/username_бота
```

`BOT_TOKEN` нужен для запуска бота.
`CONTACT_PROFILE_URL` используется в кнопке «Связаться с сотрудником».
`BOT_LINK` подставляется в текст для соцсетей.

Из папки проекта `zoo-totem-bot/` выполните:

```bash
python -m bot.main
```

## Где менять вопросы и тексты

Весь контент викторины хранится в файле:

```text
bot/quiz_data.py
```

Там можно редактировать:

- приветственные и служебные тексты в `TEXTS`;
- подписи кнопок в `BUTTONS`;
- порядок животных в `ANIMAL_ORDER`;
- вопросы и ответы в `QUESTIONS`;
- описания результатов и изображения в `ANIMALS`;
- шаблон текста для соцсетей в `SHARE_TEXT_TEMPLATE`.

## Как работает алгоритм подсчёта

У каждого варианта ответа есть словарь `scores`.

```python
"scores": {"otter": 2, "lemur": 1}
```

Когда пользователь выбирает ответ, бот прибавляет баллы указанным животным. В конце выбирается животное с максимальным количеством баллов. Если несколько животных набрали одинаковый максимум, побеждает животное, которое стоит раньше в `ANIMAL_ORDER`.

## Как заменить изображения

Положите реалистичные изображения животных в папку:

```text
bot/animals/
```

Имена файлов должны совпадать с полем `image` в `ANIMALS`:

```text
alpaca.jpg
otter.jpg
meerkat.jpg
lemur.jpg
amur_tiger.jpg
```

Если изображение не найдено, бот всё равно покажет результат текстом и сообщит, какой файл нужно добавить.

## Связаться с сотрудником

После результата пользователь может нажать «Связаться с сотрудником».

Бот отправит:

- ссылку на профиль из `CONTACT_PROFILE_URL`;
- готовое приветственное сообщение;
- название животного, которое получилось в викторине.

Сообщение можно скопировать и отправить сотруднику вручную.

## Оставить отзыв

После кнопки «Оставить отзыв» бот ждёт одно текстовое сообщение. Отзыв сохраняется в файл:

```text
reviews.txt
```

Файл создаётся автоматически в корне проекта при первом отзыве.

## Если состояние пользователя сбросилось

Состояние хранится в памяти. Если бот перезапущен, текущая сессия пользователя теряется. В этом случае при нажатии старой кнопки или отправке сообщения бот объяснит, что сессия сброшена, и покажет кнопку «Начать заново».
